# Notion Document Writing

Use the official Notion MCP server configured by this plugin.

## Workflow

1. Confirm the target page, database, or parent location before writing.
2. Search for an existing page first; update it instead of creating duplicates.
3. Draft the exact content before creating or replacing shared workspace content.
4. Structure documents with context, decision or purpose, details, owner, and next action when applicable.
5. Link generated documents back to source pages or external evidence.

## Safety

- Do not create large task trees or broad page rewrites without confirmation.
- Respect Notion sharing boundaries.
- Keep assumptions explicit when source material is ambiguous.
