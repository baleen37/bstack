# Slack Search

Use the official Slack MCP server configured by this plugin.

## Workflow

1. Identify the workspace, channel, person, thread, and time range before searching.
2. Start with narrow search terms, exact phrases, and useful Slack modifiers.
3. Ask before searching private channels or broad people-related history.
4. Open relevant threads after search results; do not summarize from snippets alone.
5. Separate direct Slack evidence from your own interpretation.

## Safety

- Keep private Slack content scoped to the user's request.
- Redact secrets, customer identifiers, and personal data.
- Cite source channels, dates, and message links when available.
